# If (se) e Else (senão)
. É um controle de fluxo de condições. 
. Sintaxe:
. A sintaxe básica
. para if é:
```c
if (condição) {
    // instruções para condição verdadeira
}
```

. Para if...else :
```c
if (condição) {
    // instruções para condição verdadeira
} else {
    // instruções para condição falsa
}
```
